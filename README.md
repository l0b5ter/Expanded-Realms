# Expanded-Realms
Unofficial DLC with toggleable features


### Features:
- __Civ Starting Bonus__
  
    Rename units by right-clicking the unit name
  
- Unit Rename
- Fog Regrows
- City takes pillage damage
- Fallout damage
- Nuclear Plant Can Meltdown (currently disabled)
- Helicopters on coast
- More CS Quests
- CS Conquest Penalty
- War Workers: Roads & Forts
